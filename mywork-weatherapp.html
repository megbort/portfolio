Weather App

Live Demo View on GitHub

I decided to do this project to get more practice with APIs. Weather Apps are something we commonly find on our phones and computers, so I thought it would be fun to try and make my own.

Process

First thing I did was look online for an available API I could use. I eventually found OpenWeatherMap, a weather API that was free to use with an account. I signed up and got my API key, then coded out the HTML so I had places to show the query results.

I read over their documentation to find the endpoints and parameters I needed. I then used fetch to pull data from the API. I kept adding results to the page until I was satisfied with the weather information being displayed. Then did some QA on the search functionality and made sure it was working properly. Lastly, I styled the page with some CSS.
