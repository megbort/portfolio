Weather Application

Live Demo

I decided to do this project to get more practice with APIs. Weather Apps are something we commonly find on our phones or computers, so I thought it would be fun to try and make my own.

Process

I first looked online for available APIs I could use and found OpenWeatherMap, a weather API, that was free to use with an account. I signed up and got my API key. Next I coded out the HTML so I had places to show the query results.

I used fetch to pull data from the OpenWeather API and using their documentation I changed the parameters to get my required results. I kept adding more results to the page until I was satisfied with the weather information. Lastly, I styled the page with some CSS.
