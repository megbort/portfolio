News App

Video Demo View on GitHub

This project was completed for an assignment in my App Development Strategy 2 class in Winter 2022. The app was developed using React Native and News API. This was my second project using React, which gave me great insight into the powerful framework.

Process

This assignment consisted of 2 parts; first implementing the designing and structure, second adding in the API functionality. I started by creating some simple designs and assets (fonts, logos, etc) for the app package. I then created a blank React project, using Expo as the bundler. All testing was done using an Android Studio emulator.

After completing all my screens, components and styling, I moved on to create an API key from News API and try and get it working. On the home screen it was simple to put in parameters of just the most relevant news by current date. However, for the "search" screen I had a few issues with trying to raise the state and update the results on screen. After debugging for quite some time I was able to get the results to show up.

All of the project code can be found on the News App GitHub Project.
