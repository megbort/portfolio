Magic 8-Ball

Live Demo

This was a side project I decided to work on for fun in November 2021. I wanted to create an imitation of the popular fortune teller toy called the “Magic 8-Ball”. To play the game you simply need to ask a question, shake the 8-ball then turn it over and read the answer revealed. My idea was to bring this into a digital space using CSS animations and JavaScript.

Process

I first started out by making some simple wireframes in Adobe XD for all viewports. Then I created the 8-Ball in Adobe Illustrator. After I had the layout and assets I started coding the project. I used a simple technique for getting the fortune by using the math.random object and adding all the fortunes in an array. I also created a shaking movement with CSS keyframes.

Even though this project didn't take too long and the code wasn't overly complicated I had a lot of fun making it, which is why I've decided to feature it in my projects.
