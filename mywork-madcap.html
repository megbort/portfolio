Madcap Media Group

Live Demo

Madcap Media was a company I created for a final project in my Mobile Web Development class in August 2021. This class focused on developing websites with a mobile-first approach and making them responsive using media queries. The final project requirements were to code a responsive homepage with 5 sections using mobile-first approach and have 3 jQuery plugins integrated.

Process

I first created mock-ups for all device sizes in Figma. After I had my designs completed, I created a logo in Adobe Illustrator and collected all media and assets needed for the page.

Once everything was prepared I started coding at the 320px viewport (mobile). After I finished with this, I moved up to each viewport and added media queries to match the mock-ups. Once I had made the page responsive and added all my media queries, I integrated all of my jQuery plugins.
