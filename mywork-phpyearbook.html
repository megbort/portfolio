PHP Yearbook

Visit on Github

This was a term project for my Webscripting 2 class, completed in December 2021. The purpose of the project was to create a yearbook using dynamic data and the CRUD method. This was my first project using PHP and found it overall to be pretty fun.

Process

The project has three main pages for the user flow. These pages were mocked-up before coding to get a sense of the project.

Yearbook Cards - This is where information entered or stored in the database would display the student cards.

Login Screen - If the user wanted to edit or delete any of the cards, they would be directed to a login screen.

Edit Page - After the correct login credentials are entered, the edit page would become available. On this page the user can add new, edit, or delete student data.

To start implementing this project we first learned about mySQL and how to create and update databases. After the database was created for the yearbook I made tables for the user information and student cards. Next was to connect our database with PHP and SQL. I then created the pages to display the cards, login and edit. After testing all the functionality of the yearbook I finished off by adding the styling from the mock-ups.
