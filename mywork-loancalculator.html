JavaScript Loan Calculator

Live Demo

My goal for this project was to make a loan calculator using only vanilla JavaScript. I thought it would be good to go back to the basics on how I first started to learn JavaScript, coding calculators!

Process

I first thought about what type of calculator I wanted to make. Since I'm currently a student with student loans *soft cry* I thought it would be relevant to make a loan calculator. Next, I created some simple mock-ups for all viewports.

I then coded out all the HTML markup for where my Javascript calculations would be displayed. Afterwards, I coded the Javascript and verified the calculator was working properly. Lastly, I gave it some style and breakpoints with some CSS.
