Flashpoint Photography

Live Demo

Flashpoint Photography was my first Bootstrap website and was submitted for a final project in December 2021. The project requirements were to create a Bootstrap website but make enough changes so that it wasn't obvious it was a Bootstrap site. This gave me lots of practice with using the framework.

Process

I first made some wireframes to get a general idea of how I wanted to layout the page. While making these I was checking the Bootstrap documentation to see what features were available to me.

Afterwards, I started coding the HTML markup and Bootstrap classes. Figuring out how the classes and naming conventions worked took me the longest, but once I figured it out I was able to get the page looking how I wanted. I finished it off by adding some additional CSS to change the font, button colours and a few other things to disguise Bootstraps default styles.
